# Finance Dashboard UI

A clean and interactive frontend-only finance dashboard built to visualize financial data, manage transactions, and demonstrate role-based UI behavior.

---

# Features

* Dashboard Overview**

  * Total Balance, Income, Expenses
  * Time-based and category-based charts

* Transactions**

  * View, search, filter, and sort transactions
  * Displays date, amount, category, and type

* Role-Based UI (Simulated)**

  * Viewer → read-only access
  * Admin → add/edit transactions

* Insights**

  * Spending patterns
  * Monthly comparisons
  * Key observations

---

## Tech Stack

* React.js
* Tailwind CSS (or your styling choice)
* Chart library (Chart.js / Recharts)

---



## Approach

* Component-based architecture
* Managed state for transactions, filters, and roles
* Used mock data for simulation
* Focused on clean UI, responsiveness, and usability

---

## Notes

* No backend used (frontend-focused project)
* Role-based behavior is simulated
* Designed for clarity, simplicity, and scalability
